# Projeto CI/CD com GitHub Actions

## 📚 O que é este projeto?
É um trabalho da faculdade para aprender a automatizar testes usando GitHub Actions.

## 🎯 O que faz?
- Roda testes automáticos quando você faz upload do código
- Pode ser executado manualmente
- Roda sozinho em horários programados
- Gera relatórios bonitinhos
- Guarda os relatórios para você ver depois

## 🚀 Como usar?
1. Faça upload do código para o GitHub
2. A pipeline vai rodar sozinha
3. Veja os resultados na aba Actions

## 📊 Relatórios
Os relatórios ficam salvos como "artifacts" e também no GitHub Pages.

## 🔧 Tecnologias
- GitHub Actions
- Python
- Pytest
- GitHub Pages
